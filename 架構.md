# 架構

## 分層

- Tier: 用來做`系統`分層

- Layer: 用來做`軟體`分層



|       | View           | Business Logic | Data Access    |
| :---- | :------------- | :------------- | :------------- |
| Tire  | Client         | Server         | Database       |
| Layer | View           | View           | View           |
| Layer | Business Logic | Business Logic | Business Logic |
| Layer | Data Access    | Data Access    | Data Access    |
