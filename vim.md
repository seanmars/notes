# Vim Cheat Sheet

## Editor mode
- `esc` to command mode.

## Command mode
- `i` to insert mode.
- `yy` to copy current line.
- `dd` to delete current line.
- `p` to paste the copied or deleted text `after` the current line.
- `P` to paste the copied or deleted text `before` the current line.
- `/` search
- `n` search next
